What components make a computer work?

The basics to internal components and computer software

  • Operating Systems?

    Operating Systems are a type of system software that manage all computer resources.
    They act as an interface between software and hardware.
    The 3 most common Operating Systems are Windows, macOS, and Linux.

  • Software?

    Software refers to instructions, data, or programs used to operate computers.
    Examples include Spotify, ChatGPT, and WhatsApp.

  • Hardware?

    Hardware includes physical components like CPUs, motherboards, and GPUs.
